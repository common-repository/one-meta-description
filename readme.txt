== Description ==

Show meta description in your homepage, single post and page. This plugin helps you to attract visitors and boost traffic. Just plug and play. (Wordpress Simple SEO - Meta Description).

If you want to know how this plugin works, right click on your page and then select view page source. You will see your meta description between the header tag.

* <a href="http://wp.labnul.com/plugin/one-meta-description/">Plugin Homepage</a>

== Screenshots ==

1. Right click on your browser and then select view page source

2. You will see your meta description 

== Installation ==

1. Extract the plugin
2. Upload 'one-meta-description' folder to the '/wp-content/plugins/' directory, 
3. Activate the plugin through the 'Plugins' menu in WordPress
